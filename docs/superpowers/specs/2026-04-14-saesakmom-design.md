# 새싹맘 — 디자인 스펙

## 개요

육아 커뮤니티 + 육아용품 비교 서비스. 임산부 및 영유아(0~3세) 초보맘 대상.

## 기술 스택

- **프론트엔드/백엔드**: Next.js 16 (App Router) + TypeScript + Tailwind CSS 4
- **ORM**: Prisma
- **DB**: Supabase PostgreSQL
- **인증**: NextAuth (이메일/비밀번호 + 카카오 소셜 로그인)
- **배포**: 추후 결정

## 기능 1: 육아 커뮤니티

### 게시판 구조

- 관리자가 생성하는 카테고리 게시판 (임신, 출산, 육아일상, 수유/이유식, 건강, 자유게시판 등)
- 게시글에 태그 추가 가능 (예: #이유식 #산후조리)
- 카테고리 + 태그 혼합 방식으로 탐색

### 핵심 기능

- **게시글**: 작성, 수정, 삭제, 이미지 첨부
- **인기글**: 좋아요/조회수 기반 인기글 선정
- **댓글**: 대댓글 지원
- **신고**: 게시글/댓글 신고 → 관리자 검토
- **검색**: 제목/내용/태그 기반 검색

### 사용자 프로필

- 닉네임, 프로필 이미지
- 내가 쓴 글, 댓글, 좋아요한 글 조회

## 기능 2: 육아용품 비교

### 운영 흐름

1. 관리자가 제품명 + 카테고리 입력
2. AI가 온라인에서 제품 정보 자동 수집
3. AI가 리뷰 크롤링 후 긍정/부정 분류
4. 결과를 양식에 맞춰 자동 저장
5. 관리자 검수 후 게시

### 제품 정보 양식

#### 기본 정보

| 항목 | 설명 |
|------|------|
| 제품명 | 정식 모델명 |
| 브랜드 | 제조사/브랜드명 |
| 카테고리 | 유모차 / 카시트 / 젖병 / 아기띠 등 |
| 가격대 | 최저가 ~ 최고가 |
| 출시일 | 출시 연도/월 |
| 제품 이미지 | 대표 이미지 |
| 대상 연령 | 사용 가능 개월 수 |
| 원산지 | 제조국 |

#### 카테고리별 상세 스펙

**유모차**: 본체 무게, 접었을 때 크기, 접이 방식(원터치/양손), 바퀴 타입(에어/우레탄), 양대면 전환 여부, 리클라이닝 각도, 선바이저 확장, 하부 바구니 용량, 최대 하중

**카시트**: 사용 연령/체중 범위, 장착 방식(ISOFIX/벨트), 회전 여부(360도/고정), 측면 보호(SIP), 리클라이닝 단계, 헤드레스트 높이 조절, 본체 무게, 커버 탈착/세탁 여부

**젖병**: 용량(ml), 소재(PPSU/유리/실리콘), 꼭지 단계, 배앓이 방지 구조, 호환 꼭지, BPA-free 여부, 세척 편의성, 내열 온도

**아기띠/힙시트**: 사용 개월 수, 최대 하중, 착용 방식(앞/뒤/옆), 힙시트 분리 여부, 폴딩 여부, 본체 무게, 통기성/메쉬 여부, 허리벨트 사이즈

#### 안전 및 인증

| 항목 | 설명 |
|------|------|
| KC 안전인증 | 인증 번호 및 등급 |
| 해외 인증 | EN(유럽), ASTM(미국), SG(일본) 등 |
| 유해물질 검사 | 환경호르몬, 중금속 검출 여부 |
| 리콜 이력 | 리콜 여부 및 내용 |

#### AI 리뷰 분석

**쇼핑몰 리뷰 (네이버 쇼핑 + 쿠팡)**:
- 긍정 리뷰: 주요 장점 키워드 TOP 5, 대표 긍정 리뷰 3개, "이런 분에게 추천"
- 부정 리뷰: 주요 단점 키워드 TOP 5, 대표 부정 리뷰 3개, "이런 분은 비추"
- 총 리뷰 수, 평균 별점, 분석 일자

**맘카페 후기 (별도 섹션)**:
- 네이버 카페 기반 후기 수집
- 쇼핑몰 리뷰와 분리하여 별도 표시

#### 비교 포인트

| 항목 | 설명 |
|------|------|
| 가성비 점수 | 가격 대비 스펙/리뷰 종합 (AI 산출) |
| 인기도 | 리뷰 수 + 커뮤니티 언급 빈도 |
| 유사 제품 | 같은 가격대/카테고리 경쟁 제품 목록 |
| 종합 추천 점수 | 스펙 + 리뷰 + 안전인증 종합 (AI 산출) |

### 초기 카테고리

유모차, 카시트, 젖병, 아기띠/힙시트 (이후 점차 확장)

## 기능 3: 관리자 시스템

- **제품 관리**: 제품 추가/수정/삭제, AI 정보 수집 트리거, 검수 후 게시
- **커뮤니티 관리**: 신고 접수 목록, 게시글/댓글 삭제, 사용자 제재(경고/정지/영구밴)
- **운영 대시보드**: 가입자 통계, 인기 제품/게시글, 트래픽 현황

## 기능 4: 광고

- 광고 슬롯을 배너/제품 페이지 등에 미리 배치
- 초기에는 비활성 상태, 광고주 확보 시 활성화
- 관리자가 광고 on/off 제어

## 비기능 요구사항

- **모바일 퍼스트**: 모바일 최적화 우선, 데스크톱도 지원하는 반응형
- **추후 웹앱**: PWA 등으로 전환 가능하도록 고려
- **인증 방식**: 이메일/비밀번호 + 카카오 소셜 로그인
